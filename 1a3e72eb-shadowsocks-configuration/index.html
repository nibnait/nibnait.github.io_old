工具配置笔记

Shadowsocks配置

转载自红宝书使用教程

#windows

  1. 下载客户端：shadowsocks.zip
    解压到电脑桌面，然后双击运行shadowsocks.exe

  1. 设置说明:
    鼠标右键点击小飞机图标，编辑服务器IP、端口、密码…

  2. 点击确定，右键小飞机图标，点击“启用系统代理”

系统代理模式：默认是PAC模式（建议使用）

PAC：智能代理，国内网站正常访问。

全局：不管你访问国内还是国外全部走代理（这样访问国内就慢了，要绕道）。

开机启动：推荐

编辑PAC文件：它是被屏蔽网站列表，可自己添加或删除代理网址。

#Mac

  • 下载、安装客户端ss-2.6.3.dmg


（请把小飞机拖到应用程序文件夹里）

点击小飞机软件运行，设置方式参考上面。

  • 填写账号后，点击电脑屏幕右上角飞机图标－－点击“服务器”－－勾选你设置好的服务器。

#Android

  • 手机扫描二维码下载客户端（微信扫一扫，点击右上角，选择在浏览器中打开链接）


点击直接下载ss-2.10.3.apk

  • 填写：服务器地址、远程端口、加密方式（邮件有）、本地端口（1080），点击右上角开启代理。

路由：绕过局域网和中国大陆

一次性认证：关

#iPhone\iPad

  • （提醒！Surge目前下架，该App上架后建议使用自己的苹果账号购买）

打开手机“设置－－iTunes与App Store

  • 点击Apple ID，点击注销，登录共享账号（密码 2a3b4c5D）：

  • 打开App Store，点击右下角“更新”－“已购项目”

  • 下载Surge-Web Dveleper Tool，下载后请切换回你自己的苹果账号。

  • 打开Surge，点击右上角

  • 点击Download Configuration from URL

  • 输入https://wasai.org/ss.conf点击OK

  • 点击右上角Done

  • 点击Edit

  • 点击Proxy，进入里面

  • 修改成你的账号（Server: 服务器IP、Port: 服务器端口、Password: 密码）

  • 修改后，点击右上角Done保存。

点击Setup

  • 点击Download

  • 点击Allow

  • 按指纹授权

  • 再次点击Start，屏幕顶部会出现“VPN”图案，代表设置成功了。

不需要用的时候，点击Stop关闭即可。
